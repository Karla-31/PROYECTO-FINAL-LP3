Listado de 102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<html> Define el inicio y fin de un documento HTML. <html>...</html> Es el contenedor principal.
<head> Contiene metadatos y enlaces externos. <head></head> No se renderiza en la página.
<body> Contiene el contenido visible de la página. <body></body> Se utiliza junto con <html>.
<title> Define el título de la página. <title>Mi Página</title> Se muestra en la pestaña del navegador.
<h1>-<h6> Define encabezados jerárquicos. <h1>Título</h1> <h1> es el más importante, <h6> el menos.
<p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puede contener otros elementos en línea.
<a> Define un enlace. <a href="https://example.com">Enlace</a> Usar target="_blank" para abrir en otra pestaña.
<img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción"> Usar atributo alt para accesibilidad.
<div> Define un contenedor. <div>Contenido</div> Usado para estructurar elementos.
<span> Define un contenedor en línea. <span>Texto</span> Útil para aplicar estilos específicos.
<ul> Define una lista desordenada. <ul><li>Elemento</li></ul> Usa <li> para los ítems.
<ol> Define una lista ordenada. <ol><li>Elemento</li></ol> Genera una lista numerada.
<li> Define un elemento en una lista. <li>Elemento</li> Usado con <ul> o <ol>.
<table> Define una tabla. <table>...</table> Debe contener <tr> y <td>.
<tr> Define una fila de tabla. <tr><td>Celda</td></tr> Va dentro de <table>.
<td> Define una celda de tabla. <td>Contenido</td> Va dentro de <tr>.
<th> Define una celda de encabezado. <th>Encabezado</th> Suele estar en la primera fila.
<form> Define un formulario. <form></form> Para entradas y botones.
<input> Crea un campo de entrada. <input type="text"> Usar diferentes tipos como email, password.
<button> Define un botón. <button>Enviar</button> Puede ser tipo "submit" o "reset".
<textarea> Crea un campo de texto multilínea. <textarea>Texto</textarea> Especificar filas y columnas.
<label> Asocia un texto con un campo. <label for="nombre">Nombre</label> Usar atributo "for".
<select> Crea un menú desplegable. <select><option>Opción</option></select> Contiene etiquetas <option>.
